> Statement Nr. 3:
>
> „Wir können doch schon Cloud–wir sind cloud-ready!“! Da man die Anwendungsschicht auf einerVirtuellen

> Skalierung einzelner Komponenten, so die Bereichsleitung. Wie stehen sie zu diesem Urteil?
